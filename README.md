# escape from school

Unity 6000.5.1f1 기반 2D 탑다운 탈출 게임 초안입니다.

## 실행

1. Unity Hub에서 이 폴더를 프로젝트로 추가합니다.
2. `Assets/Scenes/SampleScene.unity`를 엽니다.
3. Play를 누르면 게임 오브젝트와 UI가 자동 생성됩니다.

## PC 조작

- WASD 또는 방향키: 이동
- `E`: 다음 구역으로 이동
- `Space`: 공격

## 현재 구성

- 배경: 순천 복성 고등학교 콘셉트의 임시 2D 맵
- 방식: 탑다운 2D 탈출 게임
- 진행: 옥상(5층) -> 물리실/지구과학실(4층) -> 화학실/창의융합실(3층) -> 도서관(3-2층) -> 학년실/전산실(2층) -> 가정실/생태농장(1층) -> 운동장 미로
- 목표: 모든 구조 대상자를 구하고, 각 구역 보스를 처치한 뒤 운동장 미로를 통과해 탈출
- 실패 조건: 체력 0 또는 구조 대상자를 모두 구하지 못한 상태로 마지막 출구 도착
- 특수 이벤트: 가정실 보스는 플레이어와 보스를 생태농장으로 순간 이동시킴
- 추후 실제 학교 정보와 에셋이 들어오면 `SchoolEscapeGame`의 스테이지 데이터를 기준으로 교체하면 됩니다.

## 맵 제작 순서

1. 각 구역을 종이에 직사각형으로 먼저 그립니다.
2. 출발점, 출구, 구조 대상자, 아이템, 보스 시작 위치를 표시합니다.
3. Unity에서는 처음에 사각형 오브젝트로 복도와 방을 배치합니다.
4. 플레이 테스트로 길이 막히는지, 보스가 따라오는지, 출구까지 도달 가능한지 확인합니다.
5. 동선이 확정되면 교실/복도/운동장 이미지를 실제 에셋으로 교체합니다.
